# TextStore

Whether to store message content as is (plaintext), as a hashed value (sha256) or not at all (nostore)

## Enum

* `NOSTORE` (value: `'nostore'`)

* `PLAINTEXT` (value: `'plaintext'`)

* `SHA256` (value: `'sha256'`)
